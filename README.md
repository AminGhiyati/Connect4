# 🎮 Connect Four Multiplayer Game with Chat

This project is a browser-based **Connect Four** game for **two players**, featuring real-time communication via **WebSockets (Socket.IO)**. It also includes a built-in **live chat**. The application is modular and uses **Vite** as the build tool.

## 🚀 Features

- ✅ Multiplayer support via **WebSockets (Socket.IO)**
- ✅ **Drag & Drop** functionality for making moves
- ✅ **Player login** with custom name
- ✅ Funny **welcome message** from JokeAPI (with fallback in case of API failure)
- ✅ **Visual game board** using SVG/HTML
- ✅ **Real-time chat** between connected players
- ✅ **Win detection** and winner announcement with sound effect
